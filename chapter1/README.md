## 什么是编程

编程是用计算机语言告诉计算机如何完成特定任务的过程。就像我们用自然语言与人交流一样，我们用编程语言与计算机交流。

## 为什么选择 AI 辅助编程

- **降低学习门槛**：AI 可以帮助解释复杂概念
- **提供实时帮助**：遇到问题时立即获得指导
- **加快开发速度**：自动化常见任务
- **最佳实践指导**：学习行业标准的编码方式
